# Развертывание приложения

## ⚠️ Важное замечание

Ваше приложение использует API роуты Next.js. Для корректной работы всех функций рекомендуется использовать **Vercel**.

## Рекомендуемый способ: Vercel

### Преимущества Vercel для вашего приложения:
- Полная поддержка Next.js API роутов
- Автоматическое развертывание
- Бесплатный план
- Лучшая производительность для Next.js приложений

### Развертывание на Vercel:

1. **Установите Vercel CLI:**
```bash
npm install -g vercel
```

2. **Войдите в аккаунт:**
```bash
vercel login
```

3. **Разверните приложение:**
```bash
vercel
```

4. **Следуйте инструкциям в терминале**

### Альтернативный способ через веб-интерфейс:
1. Зайдите на [vercel.com](https://vercel.com)
2. Нажмите "New Project"
3. Подключите ваш GitHub/GitLab/Bitbucket репозиторий
4. Vercel автоматически определит настройки Next.js
5. Нажмите "Deploy"

## Альтернативный способ: Netlify

Если вы хотите использовать Netlify, нужно будет переписать API роуты как Netlify Functions.

### Подготовка для Netlify:

1. **Используйте статический экспорт (ограниченная функциональность):**
   - Переименуйте `netlify.toml` в `netlify-static.toml`
   - Используйте `netlify-functions.toml` для полной функциональности

2. **Для полной функциональности с API:**
   - Создайте Netlify Functions для каждого API роута
   - Обновите fetch запросы в приложении

### Развертывание на Netlify:

#### Способ 1: Через Netlify CLI
```bash
npm install -g netlify-cli
netlify login
netlify init
```

#### Способ 2: Через веб-интерфейс
1. Зайдите на [netlify.com](https://netlify.com)
2. Нажмите "New site from Git"
3. Подключите репозиторий
4. Настройте параметры сборки:
   - Build command: `npm run build`
   - Publish directory: `out` (для статического) или `.next` (для функций)

## Проверка развертывания

После развертывания проверьте:
1. ✅ Главная страница загружается
2. ✅ API запросы работают
3. ✅ Формы отправляются
4. ✅ Навигация работает

## Проблемы и решения

### Если API не работает на Netlify:
- Переключитесь на Vercel
- Или создайте Netlify Functions

### Если сборка не проходит:
1. Проверьте Node.js версию (18+)
2. Убедитесь, что все зависимости установлены
3. Проверьте логи сборки

### Если сайт не отображается:
1. Проверьте настройки публикации
2. Убедитесь, что файлы собраны правильно
3. Проверьте консоль браузера на ошибки
